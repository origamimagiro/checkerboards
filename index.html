Website for listing square origami checkerboard models folded from a two-color square, ranked by efficiency. This website augments the listing found at Peter Budai's website. Crease patterns are provided in .fold format where available, reproduced with permission from the authors.

Folded models are typed as either seamless (each color-change region is formed by a visible subset of a continuous crease pattern face, same as "clean" in Budai's listing), fuzzy (i.e., not seamless), or flippable (each grid cell can be flipped locally between white/color states via a single fold).

The year column represents the year the model was designed. If multiple models were designed in the same year by the same author, we distinguish them via their number in the "#" column.

Efficiencies are listed as the ratio of the target grid side to to the side length of the starting square. Exact algebraic efficiencies are provided where available. Models that are most efficient for a particular grid size and type are highlighted in white. When evaluating efficiencies between types, seamless and flippable types both count as fuzzy.
